# NLUX

NLUX is een Collections Discovery platform voor de Nederlandse Cultureel Erfgoed sector, gebouwd op [Project LUX](https://lux.collections.yale.edu/) van Yale University en de open [Linked.art](https://linked.art/) standaard (gebaseerd op CIDOC-CRM).

## Missie

NLUX maakt Nederlandse erfgoedcollecties vindbaar, verbindbaar en open — voor onderzoekers, instellingen en het publiek — via gedeelde open-source infrastructuur gebouwd op internationale linked data standaarden.

## Status

Het project bevindt zich momenteel in **Phase 0 — Foundation** (Q2 2026).

Zie de [Roadmap](NLUX-roadmap.md) voor een volledig overzicht van mijlpalen en planning.
